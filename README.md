🚀 Capacitação em Síntese RTL

📌 Sobre o Projeto

Este repositório concentra os projeto, exercicios e implementações desenvolvidos durante a capacitação em Sintese RTL (Register Transfer Level), realizada por meio da Softex.
O objetivo principal foi aplicar fundamentos de projeto de sistemas digitais, incluido descrição em nivel de transferência de registradores, simulação e sinteses de circuitos, com foco em boas práticas de desenvolvimento em hardware.
